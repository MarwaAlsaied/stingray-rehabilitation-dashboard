rules for scoring in this prototype:

For each exercise:
collect pitch/roll/yaw values while user performs it
calculate:
- pitch_range = max(pitch) - min(pitch)
- roll_range = max(roll) - min(roll)
- yaw_range = max(yaw) - min(yaw)
- stability = average change between readings

Then score each exercise out of 3:

3 = good range + stable movement
2 = acceptable range
1 = small movement or unstable
0 = no meaningful movement

Fast prototype logic:

movement_range = largest of pitch_range, roll_range, yaw_range

if movement_range >= 45 and stability <= 8:
    score = 3
elif movement_range >= 30:
    score = 2
elif movement_range >= 15:
    score = 1
else:
    score = 0

total_score = exercise1 + exercise2 + ... + exercise7
